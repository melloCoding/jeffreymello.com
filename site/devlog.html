Clothes Conundrum

Clothes Conundrum is a game that was originally made for the Pirate Software Game Jam 2024 but I was unable to finish the game in time for the end of the game jam. So why let the hard work go to waste. The game is about a wizard who had to go around his home picking up clothes, but if you dont pick the clothes up fast enough they will multiply.
